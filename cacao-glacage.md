# 🍫 Cacaoglacage (zonder glucose)

Eindproduct ≈ 500g (er verdampt zeg 16 g water in het proces)

Bewaren:
- vriezer: tot 3 maanden (verpak luchtdicht, in porties)  
- koelkast: 5–7 dagen (verpak luchtdicht, liefst in kleine bak)  

|        | Ingredient   | Hoeveelheid |
| ------ | ------------ | :---------: |
| **Deel 1** |          |             |
|        | Room         |    110 g    |
|        | Suiker       |     85 g    |
|        | Water        |     40 g    |
|        | Citroensap<br>(of 1,33 g citroenzuur) | 4 g |
| **Deel 2** |          |             | 
|        | Cacaopoeder (10–12% vet) | 60 g |
|        | Suiker       |     80 g    |
|        | Water        |     80 g    |
| **Deel 3** |          |             |
|        | Gelatine     |      8 g    |
|        | Water (koud) |     40 g    |

---

## Werkwijze

- 1️⃣ **Gelatine weken** (10 minuten weken in koud water (40 g)), en opzij zetten
- 2️⃣ **Invertsuiker maken** (laag vuur)
  - Room + suiker + water + citroensap in een steelpan  
  - **Zachtjes verwarmen 3–4 min**, niet koken  
  - Even roeren tot suiker volledig is opgelost  
  - Dit vormt de glansgevende siroop
- 3️⃣ **Cacao-siroop**
   - Cacaopoeder + suiker + water mengen tot glad papje  
   - Geen klontjes

- 4️⃣ **Samenvoegen en koken**
  - Voeg cacao-siroop langzaam toe aan de warme invertsuiker  
  - Meng goed met garde  
  - Breng **alleen nu** aan de kook tot **104 °C** (viscositeit en glans)  
- 5️⃣ **Gelatine toevoegen**
  - Haal van vuur en laat wat afkoelen  
  - Voeg de uitgeknepen gelatine toe en roer tot volledig is opgelost  
- 6️⃣ **Zeven en afkoelen**
  - Druk door een fijne (nylon/metalen) zeef → geen minuscule klontjes  
  - Laat afkoelen tot **32–35 °C** om direct te verwerken, of lager voor bewaren.

## Ontdooien en Verwarmen

- 1️⃣ Voorbereiden
  - Uit vriezer: eerst volledig ontdooien in de koelkast.
  - Haal glacage uit koelkast.  
- 2️⃣ Verwarmen
  - Zachtjes au bain-marie of in korte intervallen in de magnetron.  
  - Doeltemperatuur: ±32–35 °C, **niet boven 40 °C** (behoud glans/elasticiteit).

## Verwerken
- 1️⃣ Glacage heeft ±32–35 °C, **niet boven 40 °C** (behoud glans/elasticiteit).
- 2️⃣ Testen van viscositeit (zonder eindproduct)
  - Lepeltest: dip een lepel in de glacage en laat een beetje terug in de kom lopen.  
    - **Perfect:** het loopt langzaam en vloeiend, vormt een glad lint dat langzaam verdwijnt.  
    - **Te dik:** loopt nauwelijks, blijft opstapelen of trekt draden.  
    - **Te dun:** loopt te snel en dun, geen vorming van lint.  
  - Pas zo nodig aan:  
    - **Te dik:** voeg 2–5 g water per 100 g glacage, goed roeren.  
    - **Te dun:** iets laten afkoelen tot gewenste giettijd.
- 3️⃣ Toepassen
  - Glacage over taart of dessert gieten bij ±32–35 °C.  
  - Laat vanzelf **glad uitvloeien**, gebruik eventueel een paletmes voor kleine correcties.  
  - Zorg dat taart koud is → glacage zet sneller en behoudt glans.
